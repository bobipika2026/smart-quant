# Smart Quant 开发计划

## 开发原则

- 小步快跑：每个版本 ≤3 个需求
- 测试驱动：每个版本必须完全测试通过
- 部署验证：验证通过后才进入下一版本
- 代码合并：每个版本上线后提交代码
