# Smart Quant 开发计划

## 开发原则

- 小步快跑：每个版本 ≤3 个需求
- 测试驱动：每个版本必须完全测试通过
- 部署验证：验证通过后才进入下一版本
- 代码合并：每个版本上线后提交代码

---

## v0.1.0 - 基础框架

**目标**：项目能跑起来，基础功能可用

| 需求 | 描述 | 优先级 |
|------|------|--------|
| R1.1 后端启动 | FastAPI后端能正常启动，API文档可访问 | P0 |
| R1.2 前端启动 | Vue前端能正常启动，页面可访问 | P0 |
| R1.3 股票搜索 | 输入代码/名称搜索股票，显示实时行情 | P0 |

**验收标准**：
- [ ] 后端启动成功，访问 http://localhost:8000/docs 显示API文档
- [ ] 前端启动成功，访问 http://localhost:3000 显示首页
- [ ] 搜索"平安"能返回平安银行等股票信息
- [ ] 前后端联调通过

---

## v0.2.0 - 策略回测

**目标**：能运行策略回测，查看结果

| 需求 | 描述 | 优先级 |
|------|------|--------|
| R2.1 策略列表 | 显示内置策略列表 | P0 |
| R2.2 回测运行 | 选择策略+股票，运行回测 | P0 |
| R2.3 结果展示 | 显示回测结果（收益、回撤等） | P0 |

---

## v0.3.0 - 监控通知

**目标**：能监控股票，接收信号

| 需求 | 描述 | 优先级 |
