# Smart Quant 开发计划

## 开发原则

- 小步快跑：每个版本 ≤3 个需求
- 测试驱动：每个版本必须完全测试通过
- 部署验证：验证通过后才进入下一版本
- 代码合并：每个版本上线后提交代码

---

## v0.1.0 - 基础框架 ✅ 已完成

**目标**：项目能跑起来，基础功能可用

| 需求 | 描述 | 状态 |
|------|------|------|
| R1.1 后端启动 | FastAPI后端能正常启动，API文档可访问 | ✅ |
| R1.2 前端启动 | Vue前端能正常启动，页面可访问 | ✅ |
| R1.3 股票搜索 | 输入代码/名称搜索股票，显示实时行情 | ✅ |

**附加功能**：

## v0.2.0 - 策略回测 ✅ 已完成

**目标**：能运行策略回测，查看结果

| 需求 | 描述 | 状态 |
|------|------|------|
| R2.1 策略列表 | 显示内置策略列表 | ✅ |
| R2.2 回测运行 | 选择策略+股票，运行回测 | ✅ |
| R2.3 结果展示 | 显示回测结果（收益、回撤、曲线） | ✅ |

**内置策略**：

- MACD策略 (macd)
